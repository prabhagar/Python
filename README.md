## Today's Goal

For Day 10, learn these basics:
- `return` to send a value back from a function
- storing the returned value in a variable
- using the returned value in math

## How to Use This Workspace

1. Open today's class file.
2. Read the comments at the top.
3. Run the file.
4. Change the values.
5. Complete the small exercises at the bottom.

## Run a Lesson

On macOS, run:

```bash
python3 class_10_function_return_values.py
```

Or in VS Code, open any class file and run the task named `Run current Python file`.

## Beginner Rules

- Type code yourself.
- Change one thing at a time.
- If you see an error, read it slowly.
- Small progress every day is enough.
